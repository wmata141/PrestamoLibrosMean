Sistema Biblioteca de Mambot

Prestamo de Libros Desarrollado sobre el Conjunto MEAN (MongoDB Express.js Angular.js Nod.js)

Para la puesta en marcha del sistema:

1) descargue el proyecto 
2) inicie el servidor de mongodb
3) inicie su consola oprimiendo tecla (windows + r ) 
4) selecione cmd
5) posicione en la carpeta Api-rest y escriba npm install y presione (ENTER)
6) repita los pasos 4 y 5 en la carpeta Angular5
